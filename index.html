ReportTable.js

A JQuery plugin built for displaying fuller screen tables

View on Github

Caution: This plugin is under active development and should not be used in a production environment just yet.

How to use

Include the JS File: 
<script type="text/javascript" src="reporttable/dist/reporttable.min.js"></script>
Then Apply the reportTable extension to the table 
$(".reportTable table").reportTable({
     //options go here
});
Key Default Value Description
offsetHeaderHeight 0 The Offset height for the header, used if you have a fixed header on the page
offsetFooterHeight 0 The Offset height for the header, used if you have a fixed header on the page
cssPrefix reporttable the CSS prefix attached to the reportTable
headerCell th either "th" or "td", see what is used in your table header
scrollContainer null A jQuery object of the scroll container. If your table does not fill the screen and can not use the window's scroller, use this to have a scroll container
